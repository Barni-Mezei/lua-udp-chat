# Usage

## Starting the program
By running the `main.lua` you can start chatting.
Arguments:
- `-p` Set port (server's port)
- `-a` Set address (server's address)
- `-u` Specify username
- `-s` Run as a server (Starts a broadcast echo server, listening on all IPs and ports. The port cna be sepecified using `-p`)

## Using the program
In client mode, you just have to type something and after pressing enter, it will be sent to the server
which will broadcast it back to every other connected client
